Pasos Para Prácticas
Clonar la practica 
git clone la practica

cd practicaN

git branch work
git checkout work

1-Desde la Raíz de la carpeta hacer en terminal 
~/prácticaN $ mvn test

2-Una vez que todo está bien accionar el comando
git commit . -m "Ya me salió la practicaaaaa"

3- Ejecutar 
mvn install

4- Probar la Práctica 
java -jar target/practicaN.jar

 5- Generar Archivo Diferencial con la rama principal 
git diff main > ~/319053315.diff

6- Limpiar rama de trabajo
 git clean -x -f -d .
 
7- Crear una rama de prueba para probar el archivo diferencial 
git checkout main 

git branch test 

git checkout test

8 - Aplicar Parche a la rama de prueba 
patch -p1 < ~/319053315.diff

9- Verificar que todas las pruebas unitarias pasen
mvn test

10- Ejecutar el programa de la práctica 
mvn install 

java -jar target/practicaN.jar
